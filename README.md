# PrivateBrowsingOnVS
The arguments required to browse privately / incognito in Visual Studio for a wide selection of internet browsers.

# Sample Browser Configurations

## Chrome
Program:
* "C:\Program Files (x86)\Google\Chrome\Application\chrome.exe"
Arguments:
* --incognito
Friendly Name:
* Google Chrome - Incognito

## Internet Explorer
Program:
* "C:\Program Files (x86)\Internet Explorer\iexplore.exe"
Arguments:
* -private
Friendly Name:
* Internet Explorer - InPrivate

## Microsoft Edge
Program:
* "C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe"
Arguments:
* --inprivate
Friendly Name:
* Microsoft Edge - InPrivate

## Firefox
Program:
* "C:\Program Files\Mozilla Firefox\firefox.exe"
Arguments:
* -private
Friendly Name:
* Firefox - Private

## Firefox Developer Edition
Program:
* "C:\Program Files\Firefox Developer Edition\firefox.exe"
Arguments:
* -private
Friendly Name:
* FirefoxDeveloperEdition - Private

## Opera
Program:
* "C:\Program Files\Opera\launcher.exe"
Arguments:
* --private
Friendly Name:
* Opera Internet Browser - Private
